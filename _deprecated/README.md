## Deprecated Scripts

The scripts in this folder are deprecated and only preserved for posterity. They may or may not be run-able in their current form and, regardless, are not relevant to the data archived in the Environmental Data Initiative (EDI). **The 'actual' scripts are those in the top-level of this GitHub repository!**

See below for script explanations:

- Files starting with `manuscript-` are related to Nick Lyon's MS thesis chapter \#1
- Files starting with `original-qc` are from an earlier effort at wrangling the data from--what turned out to be--not actually the raw data
