## Eleven Years of Butterfly and Nectar-Producing Plant Community Data from Grassland Sites Managed with Pyric Herbivory 

[![Code DOI](https://img.shields.io/badge/Code-10.5281/zenodo.20329857-blue.svg)](https://doi.org/10.5281/zenodo.20329857)

The first chapter of Nick Lyon's M.Sc. thesis -- focused on the response of butterfly and nectar-producing flower communities to different combinations of prescribed fire and cattle grazing. All data were entered into Microsoft Access so the wrangling scripts in this repository process each of the three relational tables separately.
